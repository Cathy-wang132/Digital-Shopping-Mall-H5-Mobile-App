#### 3、技术栈

· uni-app

使用 uni-app 构建 H5 端页面，支持多端适配、快速开发与组件化管理，更加贴近真实移动电商项目的开发模式。

· uni-ui

UI 组件库采用 uni-ui，提供丰富的移动端 UI 组件，如轮播图、商品卡片、列表、表单等，使页面开发更快捷、更易维护。

·开发工具：HBuilderX

#### 4、网站内容（H5功能模块）

（1）商城首页

展示数码商品分类、热门推荐、轮播广告位、优惠活动等，让用户快速浏览商品。

（2）商品详情页

展示商品图片、价格、参数信息、规格选择以及购买操作，是用户了解商品的核心页面。

（3）购物车

支持商品加入购物车、数量修改、删除与结算，模拟真实购物流程。

（4）订单管理

展示用户订单的列表与详情，包括订单状态、支付情况与时间信息，便于学习订单流转逻辑。

（5）我的收藏

提供商品收藏管理功能，方便用户保存感兴趣的商品。

（6）个人中心

展示用户信息入口、订单入口、设置等常见电商个人页内容。
